## 2024-05-24 - Jetpack Compose Accessibility Improvements
**Learning:** In Jetpack Compose, setting `contentDescription = null` on icons that are accompanied by text is crucial to prevent TalkBack from redundantly announcing the icon along with the text. Furthermore, utilizing `onClickLabel` and `onLongClickLabel` within `combinedClickable` or `clickable` modifiers provides much more context to screen reader users than the default "Double tap to activate" announcement. Additionally, when dynamically assigning labels like `onLongClickLabel` based on item state (e.g., pinned vs unpinned), it's important to only assign the label and handle the action if the item supports that action, rather than globally applying a label that might be functionally dead for certain list items.
**Action:** Always verify if an icon needs a `contentDescription` or if its adjacent text suffices. Proactively implement `onClickLabel` and `onLongClickLabel` on interactive elements to describe the specific action they perform. Ensure dynamic accessibility labels accurately reflect the available actions for the specific item state.
