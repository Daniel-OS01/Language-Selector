## 2024-06-14 - Improve Icon Accessibility in Jetpack Compose
**Learning:** In Jetpack Compose, icons without explicit labels that are functional (e.g. Back buttons) need an accessible label and `contentDescription`, while icons that are purely decorative or immediately paired with adjacent descriptive text (like an app list item icon) should have `contentDescription = null` to prevent redundant TalkBack announcements. Making list items `Role.Button` and using `onClickLabel` also improves accessibility semantics.
**Action:** When adding or updating Compose icons, consider their context. Functional icons alone require valid content descriptions (using `stringResource`). List items or purely aesthetic graphics should set `contentDescription = null`. Add roles and click labels to interactive rows.
